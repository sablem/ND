Utilisation

### Pré-requis

Pour utiliser ce projet, vous devez avoir :
- Python 3.7 ou plus récent.
- pip installé.
- git pour cloner le dépôt.
- Bibliothèques Python : OpenCV, PyTorch/TensorFlow, gTTS...
